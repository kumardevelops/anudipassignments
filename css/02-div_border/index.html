CSS ch2- Basic Properties (Div, border etc)

Why is CSS ?

CSS stands for Cascading Style Sheets. It is a language used to describe the presentation of a document written in HTML or XML.

It allows developers to control the layout, appearance, and behavior of web pages.

What are the disadvances of CSS ?

CSS can be very frustrating as it sometimes does not show errors and we need to write too many lines of code to achieve simple tasks.
